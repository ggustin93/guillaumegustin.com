# Agent Authentication

guillaumegustin.be is a public portfolio website. It does not provide user
accounts, OAuth flows, or protected agent APIs.

Agents may read the public website, `llms.txt`, the API catalog, and the agent
skills index without authentication. The public contact endpoint accepts contact
form submissions, but agents should only submit a message after explicit user
confirmation.

Related resources:

- Site summary: https://guillaumegustin.be/llms.txt
- API catalog: https://guillaumegustin.be/.well-known/api-catalog
- OpenAPI description: https://guillaumegustin.be/openapi.json
- Agent skills index: https://guillaumegustin.be/.well-known/agent-skills/index.json
